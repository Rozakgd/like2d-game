# 🕹️ Like2D Game Hub (Official Demos & Community Showcase)

Welcome to the official game catalog for **Like2D** — a lightweight, snappy 2D game framework powered by SDL3 and Luau scripting. 

This repository is a central directory for ready-to-play games. **All game binaries and zip files are hosted exclusively in the Releases section!** You don't need to download or clone any source code here to play.

## 📥 How to Download & Play (Sat-Set No Setup!)

Playing games built with Like2D is extremely simple and requires zero installation or compilers:

1. Look at the right side of this screen and click on the **[Releases](https://github.com/USERNAME_LU/like2d-demo/releases)** tab.
2. Find the game bundle you want to play (for example: `hexaxe-demo.zip` or `community-game.zip`).
3. Download and extract the `.zip` file anywhere on your computer.
4. Double-click the **`Like2D.exe`** inside the extracted folder to start playing instantly! 🎮
